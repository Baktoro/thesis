- Για την ορθή λειτουργία του εργαλείου απαιτείται εγκατάσταση 
nodejs έκδοσης 18.15.0 (Σε ορισμένες άλλες εκδόσεις δεν λειτουργεί ορθά).

- Ακόμα, απαιτείται η εγκατάσταση ορισμένων βιβλιοθηκών μέσω npm (Node Package Manager).
Τις εγκαθιστούμε ανοίγοντας command prompt και τρέχοντας την ακόλουθη εντολή:
npm install express multer path fs adm-zip xml2js archiver util

- Στον φάκελο του project περιέχονται και δύο δοκιμαστικά αρχεία, ένα για το cadmos (.cdm)
και ένα για το canvas (.imscc).
